# Day 3 — Temperature & Output Variability

## Built
- temperature_experiment.ipynb (first Jupyter notebook)
- ask() helper function with a temperature parameter
- Experiment: same prompt run 3x at temp 0.0 and 3x at temp 1.0

## Learned
- Temperature controls how the model samples its next word
- Temp 0 = picks the most likely word every time → identical outputs
- Temp 1 = samples lower-probability words → varied outputs
- "Probabilistic" seen live: temp 0 gave the same tagline 3x, temp 1 gave 3 different ones
- Engineering judgment: temp 0 for testable/reproducible work (evals), higher for creative variety
- Notebooks = build-check-build-check, one cell at a time; variables persist between cells

## Bugs / Confusions
- Notebook needed Python + Jupyter VS Code extensions installed first
- AI agent (Codex) kept trying to write cells / duplicated cells — cleaned up manually

## Evidence
- Temp 0.0: "Wake up to something extraordinary." (identical x3)
- Temp 1.0: 3 distinct taglines, one nearly matching the temp-0 version
